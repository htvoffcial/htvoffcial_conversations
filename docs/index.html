お兄さんへのお便り

はるはる体操のお兄さんラジオへの質問やメッセージがあれば、ぜひお送りください！
◼︎ 送信方法
1.このサイトのフォームにタイトルと内容を入力
2.「GitHub Discussionsへ！」ボタンをクリック 3. 入力済みのフォームに飛びますので、内容を確認して投稿！
もしくは、GitHub Discussionsの「雑談」カテゴリに直接投稿♪

◼︎ 注意事項
・個人情報や誹謗中傷など、他の人が不快に感じる内容は避けてください。
・お兄さんは長い文が苦手なので、できるだけ簡潔に書いてください。
・投稿には原則、GitHubアカウントが必要です。また、投稿の内容はGitHubのガイドラインや規約が適用されます。

※入力内容は保持されます


免責事項

NHKの「たいそうのおにいさん」とは関係ありません。

プライバシーに関する選択

このサイトは、ユーザーのプライバシーを尊重し、以下のようなデータ収集を行っています。

  • アクセス解析ツールによる匿名の訪問者データの収集
  • お問い合わせフォームからのメッセージ内容の保存

このフォームでデータの収集を希望されない場合は拒否することができます。



これらのデータは、サイトの改善やユーザーサポートのために使用されますが、個人を特定できる情報は収集していません。
